This project uses a **singly linked list** to manage tasks dynamically.

### Features:
- Add tasks
- Search by task ID
- Delete tasks
- List all tasks

### Time Complexity:
| Operation | Time      |
|-----------|-----------|
| Add       | O(n)      |
| Search    | O(n)      |
| Delete    | O(n)      |
| Traverse  | O(n)      |

### Data Structure:
- Singly Linked List

### Benefits:
- Dynamic memory allocation
- Efficient for frequent insertions/deletions
